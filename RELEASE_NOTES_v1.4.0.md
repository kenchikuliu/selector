# Web Element Selector v1.4.0

This release adds a proper extension popup so the project feels like a complete browser tool instead of only a toolbar trigger.

## Highlights

- New popup UI with current-tab summary
- One-click `Launch Selector` action from the popup
- Clear messaging when the current browser page does not allow injection
- Quick links to install docs, GitHub, and the latest release

## Install

### Unpacked extension

1. Open `chrome://extensions` or `edge://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the `web-element-selector` folder
5. Click the extension icon to open the popup
6. Press `Launch Selector`

### Bookmarklet

Open `https://kenchikuliu.github.io/web-element-selector/` and drag the bookmark to your bookmarks bar.

### Packaged zip

Use the attached `web-element-selector-v1.4.0.zip` asset for manual extraction and sharing.
