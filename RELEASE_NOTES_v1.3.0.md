# Web Element Selector v1.3.0

This release finishes the rename rollout with proper extension icons and a cleaner distribution surface.

## Highlights

- Toolbar and extension management icons added
- Manifest updated to `1.3.0`
- Install page now uses the same icon asset as the extension
- GitHub Pages is configured to serve the bookmarklet install page from the renamed repository

## Install

### Bookmarklet

Open `https://kenchikuliu.github.io/web-element-selector/` and drag the bookmark to your bookmarks bar.

### Unpacked extension

1. Open `chrome://extensions` or `edge://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the `web-element-selector` folder
5. Pin the extension and click the icon on any page

### Packaged zip

Use the attached `web-element-selector-v1.3.0.zip` asset for manual extraction and sharing.
