# Web Element Selector v1.2.0

This release aligns the project name, repository name, extension metadata, and packaged download under one consistent identity: `web-element-selector`.

## Highlights

- Project and extension branding renamed to `Web Element Selector`
- Repository references updated to `kenchikuliu/web-element-selector`
- Extension manifest version bumped to `1.2.0`
- Packaged extension zip renamed to match the project and release version

## Install

### Unpacked extension

1. Open `chrome://extensions` or `edge://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the `web-element-selector` folder
5. Pin the extension and click the icon on any page

### Packaged zip

Use the attached `web-element-selector-v1.2.0.zip` asset for manual extraction and sharing.

## Notes

- The extension still uses only `activeTab` and `scripting`
- Restricted browser pages such as `chrome://` are intentionally skipped
- Bookmarklet support remains available through the included install page
